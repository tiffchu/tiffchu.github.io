NLP Tool & Analysis App with the UBC Data Science Institute

Overview

An interactive Natural Language Processing application built with Streamlit to support text preprocessing and analysis tasks such as tokenization, lemmatization, part-of-speech tagging, named entity recognition, and sentiment analysis. The overall goal is to enable non-technical users to apply mood detection, topic modelling, and text cleaning + EDA to text data. Project was part of an internship with the UBC Data Science Institute.

Tech & Approach

Developed in Python using Streamlit for the user interface, the app leverages SpaCy, BERTopic, nltk, transformers, etc. for linguistic tasks. The pipeline processes text efficiently and visualizes results directly in the browser with minimal setup.

Topic Modelling using LDA on ICU Transcripts with the UBC School of Nursing

Overview

This independent project was done with supervision and data from the UBC School of Nursing, exploring text data, model evaluation, and experiment design as part of a term-long research investigation. This project analyzes transcripts from intensive care unit (ICU) survivors using natural language processing (NLP) techniques. The aim is to identify the challenges people experience after hospitalization that might contribute to an unplanned readmission. Research project is part of a qualitative study led by Dr. Fuchsia Howard.

Approach & Methods

The work included data gathering, exploratory analysis, statistical modeling, and visualization to derive insights. The research was documented with code and interpretation in Jupyter notebooks and supported by written analysis.

Anomaly Detection in US Senate Investments

Overview

This project investigates unusual or statistically significant patterns in financial transactions associated with members of the U.S. Senate, using anomaly detection techniques to highlight outliers that may warrant closer inspection.

Methodology

The core pipeline includes data extraction and cleaning from publicly available financial disclosures, feature engineering to quantify investor behavior, and the application of unsupervised anomaly detection models hierarchical clustering and LSTM autoencoders to identify unusual activity. Results are visualized with interactive plots and documented in analysis notebooks.
